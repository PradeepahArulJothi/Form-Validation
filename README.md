# Login Form & Character Counter & Student tasks

This project consists of two mini web applications built using **HTML, CSS, and JavaScript**.

## Project 1 – Login Form

* Validates Name, Email, and Password fields.
* Displays error messages for invalid inputs.
* Enables the Submit button only when all fields are valid.

## Project 2 – Character Counter

* Counts characters in real time while typing.
* Validates that the text area is not empty.
* Restricts input to the maximum character limit.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Learning Outcomes

* DOM Manipulation
* Form Validation
* Event Handling
* JavaScript Functions
* Input Validation
